# Desafio DIO - HTML

Primeiro desafio HTML da escola DIO no bootcamp NTT DATA Diversidade em Tech.

Foi explanado brevemente sobre a história dos computadores, a interface client x server e foi feita uma introdução ao HTML.

O objetivo principal da atividade foi manipular o HTML e aplicar ativamente o conhecimento aprendido através do uso das Tags de marcação e seus atributos, trazendo um pequeno resumo do que foi tratado no curso, para isso, foi construída uma página HTML seguindo o modelo indicado pelo professor Diogo Mainardes.
